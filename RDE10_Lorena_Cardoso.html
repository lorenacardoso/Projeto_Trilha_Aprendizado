CSS-Cascading Style Sheets

O que é CSS?

Cascading Style Sheets - Folhas de Estilo em Cascata - tem por objetivo realizar a estilização de conteúdo de documentos HTML. Podem definir cores, fontes, espaçamentos, dentre outros.

Sintaxe:

seletor {

        propriedades: valor; /*comentario*/

        propriedades-subpropriedades: valor;

}

Componentes:

Seletores:
Seleciona elementos HTML com base no nome do elemento ao qual o estilo será aplicado.
Propriedades:
O aspecto específico do elemento que você quer usar como estilo.
Valores:
A configuração aplicada ao atributo.

Exemplo:

p {

        color: blue;

        text-align: center;

}

As três formas de se utilizar CSS:

Inline:
  • Os códigos são definidos dentro de uma tag HTML.
  • Afetam somente a tag em que foi definida.
Incorporado:
  • Os códigos CSS são definidos no cabeçalho, dentro da tag head.
  • Afetam somente o documento em que foi definido.
Externo:
  • Os códigos são definidos em um arquivo externo.
  • Afetam todos os documentos que estão linkados ao CSS.
