Definição de Hyperlinks

Um hiperlink é uma palavra, frase ou imagem na qual você pode clicar para ir para um novo documento ou uma nova seção do documento atual. Os hiperlinks são encontrados em quase todas as páginas da Web, permitindo que os usuários cliquem no caminho de uma página para outra. Os hiperlinks de texto geralmente são azuis e sublinhados, mas não precisam ser.
Os hiperlinks, geralmente chamados de "links", são comuns em páginas da Web, mas podem ser encontrados em outros documentos de hipertexto. Isso inclui certas enciclopédias, glossários, dicionários e outras referências que usam hiperlinks. Os links funcionam da mesma maneira que na Web, permitindo que o usuário pule de uma página para outra. Basicamente, os hiperlinks permitem que as pessoas pesquisem informações em hiper velocidade.

Tipos de links

Link interno
Um link entre duas páginas da web, em que ambas as páginas pertencem ao mesmo site, é chamado de link interno. Sem links internos, não existe site (a menos, é claro, que seja um site de uma página).
Link externo
Um link de sua página da web para a página de outra pessoa. Sem links externos, não há Web, pois a Web é uma rede de páginas da web. Use links externos para fornecer informações além do conteúdo disponível em sua página da web.
Links de entrada
Um link da página de outra pessoa para o seu site. É o oposto de um link externo.

Por padrão, os links aparecerão da seguinte forma em todos os navegadores:

  • Um link não visitado está sublinhado e em azul
  • Um link visitado está sublinhado e roxo
  • Um link ativo está sublinhado e vermelho

OBSERVAÇÃO:Os links podem, é claro, ser estilizados com CSS, para obter outra aparência!

Sintaxe

< a href="url">Texto do Link</a>
A <a> marca HTML define um hiperlink.
O href é o atributo mais importante do elemnto a, ele que indica o destino do linnk, definido em href="link".
Ja o "Texto do Link" é a parte que ficará visível ao leitor. Clicando nele, enviará a pessoa para o endereço URL especificado.

Atributo de Destino

Seguindo a sintaxe acima, por padrão, a página vinculada será exibida no navegador atual, para mudar isso usamos o atributo target ele especifica onde abrir o documento vinculado, < a href="url" href="valor">Texto do Link</a>. Ele tem os seguintes valores:

  • _self: Abre o documento na mesma janela / guia em que foi clicado
  • _blank: Abre o documento em uma nova janela ou guia

Exemplos:

_Self
< a href="url" target="_self">Texto do Link</a>
_Blank
< a href="url" target="_blank">Texto do Link</a>

Usando Imagem como link

Sintaxe

Para usar uma imagem como link, basta colocar a <img> tag dentro da <a>:

<a href="URL"target="valor">
<img src="local_da_imagem/nome_da_imagem"alt="descrição"> <\a>

Usando E-mail como link

Sintaxe

Use mailto: dentro do href para criar um link que abre o programa de e-mail do usuário (para permitir que ele envie um novo e-mail):

<a href="mailto:bla@ble.com"> Enviar e-mail</a>
